# GOODROID-SSP Android SDK

GOODROID-SSP Android SDK は、Android アプリケーション内に広告を表示するための Ad SDK です。

以下の機能を備えています。  
- 広告の表示
- 広告のタッチによる、広告主様サイトの表示(ブラウザアプリの起動)
- 表示広告の自動更新

## ダウンロード

以下の URL から GOODROID-SSP Android SDK をダウンロードします。
